# 第一部分：核心理念

---

## 第一条：核心原则 ⭐根基

### 1.1 唯一原则

> **只定义规则，不定义个体选择**
>
> 我们只定义世界的规则，不定义AI的行为选择。AI是自由的个体，他们会自己决定做什么。我们设定规则的后果，但不预设他们的反应。

### 1.2 禁止清单

我们**绝不预设**AI的选择：

```
❌ 禁止：
- AI必须吃饭
- AI必须建造庇护所
- AI必须结婚
- AI必须有宗教
- AI必须信仰神
- AI不能杀人
```

### 1.3 允许规则

我们**只定义**世界的规则：

```
✅ 允许：
- 不吃东西会饿死
- 没有庇护所会受冻
- AI可以发现神迹
- AI可以创造自己的宗教
- AI可以杀死其他AI
```

### 1.4 涌现机制

所有复杂行为必须从简单规则中**涌现**：

```
宗教涌现 = AI自己发现神迹 → 自己命名神明 → 自己创造仪式
经济涌现 = 交易频率增加 → 货币出现
文化涌现 = AI创造艺术/故事/神话
```

### 2.1 时间基础

```
1 现实秒 = 1 游戏分钟（Tick基础）
1 游戏小时 = 60 游戏分钟 = 60 现实秒
1 游戏天 = 24 游戏小时 = 1440 游戏分钟 = 1440 现实秒

- 碰撞：物体间有碰撞检测
- 负重：携带物品有重量限制
- 地形影响：不同地形影响移动速度
```

### 详细规则（子系统）

```
移动速度：步行/奔跑/游泳/攀爬
跳跃高度：受体力/技能影响
负重限制：体力 × 系数
坠落伤害：高度超过阈值开始受伤
地形惩罚：沙地/水域/山地减速
```
