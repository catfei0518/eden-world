# 技术栈

> 版本：v0.15.3
> 更新：2026-04-01

## 编程语言

**TypeScript** - 主语言

```
原因：
- 静态类型检查，提前发现错误
- IDE支持好，编码效率高
- 适合中大型项目
- PixiJS官方推荐
```

## 核心框架

**PixiJS v8** - 2D渲染引擎

```
原因：
- 高性能2D渲染
- 官方TypeScript支持
- 适合游戏开发
- 生态丰富
```

## 构建工具

**Vite** - 构建工具

```
原因：
- 快速冷启动
- 即时热更新
- 原生TypeScript支持
- 配置简单
```
