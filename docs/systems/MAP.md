

## 地形生成算法

### 噪声算法
```
Simplex噪声
- 业界标准
- 多层叠加
```

### 多层结构

```
1. 基础高度图
   - 大尺度Simplex噪声
   - 决定大陆/海洋分布

2. 细节噪声
   - 中尺度噪声
   - 决定丘陵、山脉

3. 微细节
   - 小尺度噪声
   - 决定岩石、植被

4. 气候带
   - 根据纬度/高度
   - 决定温度、降雨
```

### 噪声参数

```
频率（frequency）
振幅（amplitude）
叠加层数（octaves）
持久性（persistence）
```

### 控制因素

```
高度 → 地形类型
湿度 → 植被
温度 → 气候
```

## 地图分块

```
四叉树分块
- 自适应分块
- 内存效率高
- 适合无限世界

分块大小：
- 根节点：512×512格
- 最小区块：16×16格
- 中间层级：64×64格, 128×128格
```

## 初始出生点

```
混合方案：
- 程序生成多个候选点
- 评估生存条件
- 选择最优

必须有：
- 水源（100格内）
- 适居地形（平原/草地）
- 有资源（食物/木材）

应该没有：
- 极端气候
- 大型捕食者
- 自然灾害区
```

## 资源分布

```
混合分布：
- 矿物 → 噪声分布（矿脉）
- 植物 → 群落分布（森林）
- 动物 → 群落分布（栖息地）
- 水源 → 噪声分布（地下水位）
```

```
每个地形多个子群落：

```

## 洞穴/地下

### 洞穴生成（混合：噪声+侵蚀）

```
【主体结构】
- 噪声阈值决定是否有洞穴
- 低于阈值 = 空气/洞穴
- 高于阈值 = 岩石

【侵蚀痕迹】
- 水流路径
- 侵蚀通道
- 钟乳石（可选）

【矿脉】
- 洞穴壁上有矿物
- 增加资源价值
```

### 洞穴深度

```
【表层】0-50格
- 简单洞穴
- 基础矿物（石头、煤炭）
- 容易获取

【中层】50-150格
- 复杂通道
- 中级矿物（铁、铜）
- 需要工具

【深层】150格+
- 矿脉丰富
- 稀有矿物（金、银、宝石）
- 高风险高回报
```
