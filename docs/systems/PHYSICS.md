# 物理系统

## 概述

基础物理系统定义2D世界中的物体运动和交互规则。

## 维度

```
2D 俯视角
```


## 坐标系

```
混合坐标：
- 逻辑层：网格（AI决策）
- 渲染层：像素（流畅显示）
```


## 格子尺寸

```
64×64像素 = 1格 = 1米
```


## 移动速度

```
正常步行：3格/秒
奔跑：6格/秒
慢走：1.5格/秒
```


## 碰撞检测

```
半径碰撞：
- 人类：0.5格（半径30像素）
- 大型物体：1格
- 小型物体：0.3格
```


## 跳跃（简单）

```
跳跃高度：1格
跳跃时长：0.5秒
跳跃距离：1-2格
冷却时间：0.5秒
```


## 地形速度影响

```
平原/道路：×1.0
草地：×0.9
森林/灌木：×0.7
沙地：×0.6
沼泽/湿地：×0.5
水域（可通行）：×0.3
水域（不可通行）：阻挡
山地（可攀爬）：×0.5
山地（不可攀爬）：阻挡
```


## 负重系统

```
【手持层】
- 2个手持位
- 可以快速拿取
- 物品重量不限（但影响移动速度）

【背包层】
- 体积限制：10格
- 重量限制：20公斤
- 超重减速：每超10%速度-10%

【携带规则】
- 双手各1个手持位
- 背包可放多个物品
- 重量+体积双重限制

物品体积：1-5格
物品重量：0.5-10公斤
```


## 重力加速度

```
重力加速度：9.8 m/s²
最大下落速度：10 m/s
落地检测：到达地面停止
```


## 摩擦力

```
土地：×1.0
草地：×0.9
沙地：×0.7
泥地：×0.6
冰面：×0.3
石头：×1.0
```
