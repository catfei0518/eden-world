## 生存属性

### 1.1 力量 (Strength)

| 项目 | 值 |
|------|---|
| DNA字段 | `strength` |
| 范围 | 0.0 - 1.0 |
| 默认值 | 0.5 |

**影响：**
- 基础伤害 = 10 + 力量 × 20
- 搬运能力（待实现）
- 建造速度（待实现）

### 1.2 体质 (Constitution)

| 项目 | 值 |
|------|---|
| DNA字段 | `constitution` |
| 范围 | 0.0 - 1.0 |
| 默认值 | 0.5 |

**影响：**
- 最大生命值 = 75 + 体质 × 50
- 伤害减免 = 体质 × 10
- 抗性加成（待实现）

### 1.3 寿命 (Lifespan)

| 项目 | 值 |
|------|---|
| DNA字段 | `lifespan` |
| 范围 | 0.0 - 1.0 |
| 默认值 | 0.5 |

**基础寿命：** 40 + lifespan × 30 = 40-70岁

**生活方式影响：**

| 因素 | 效果 |
|------|------|
| 熟食比例高 | +10% |
| 净水比例高 | +10% |
| 生肉 | -5%每次 |
| 生水 | -3%每次 |
| 饥饿天数 | -2%每天 |
| 疾病 | -10%每次 |

**预期寿命公式：**
```
预期寿命 = 基础寿命 × 生活方式系数
```

### 1.4 免疫力 (ImmuneStrength)

| 项目 | 值 |
|------|---|
| DNA字段 | `immuneStrength` |
| 范围 | 0.0 - 1.0 |
| 默认值 | 0.5 |

**影响（待实现）：**
- 疾病概率
- 疾病持续时间
- 自愈概率

## 2. 生命值系统

### 1.1 生命值上限

```
最大生命值 = 75 + 体质 × 50

体质0.0 → 75 HP
体质0.5 → 100 HP
体质1.0 → 125 HP
```

### 1.2 生命值消耗

| 来源 | 消耗速度 | 备注 |
|------|---------|------|
| 口渴=0 | 2点/分钟 | 最快 |
| 饥饿=0 | 1点/分钟 | |
| 疾病 | 0.5点/分钟 | 视严重程度 |
| 战斗受伤 | 立即减少 | 攻击伤害 |

### 1.3 生命值恢复

| 来源 | 恢复量 | 条件 |
|------|--------|------|
| 进食 | +5点/次 | 吃饱喝足 |
| 休息/睡眠 | +10点/小时 | 安全环境 |
| 草药治疗 | +20点/次 | 使用草药 |
| 满状态 | +1点/小时 | 食物+水全满 |

### 1.4 生命值状态

| 生命值 | 状态 | 效果 |
|--------|------|------|
| 100-70% | 健康 | 正常活动 |
| 70-40% | 轻伤 | 移动速度-10% |
| 40-20% | 重伤 | 移动速度-30%，无法采集 |
| 20-1% | 濒死 | 只能休息，醒目提示 |
| 0% | 死亡 | 角色消失 |

## 2. 战斗伤害系统

### 2.1 基础攻击力

```
基础伤害 = 10 + 力量 × 20

力量0.0 → 10 伤害
力量0.5 → 20 伤害
力量1.0 → 30 伤害
```

### 2.2 武器伤害加成

| 武器 | 伤害加成 |
|------|---------|
| 徒手 | +0 |
| 木棍 | +10 |
| 石斧 | +20 |
| 铁剑 | +40 |

### 2.3 伤害计算

```
最终伤害 = 攻击伤害 - 防守减免
防守减免 = 体质 × 10
```

### 2.4 击杀所需次数

| 武器 | vs低体质(0.3) | vs中体质(0.5) | vs高体质(0.8) |
|------|---------------|---------------|---------------|
| 徒手 | 6-7拳 | 8拳 | 10拳 |
| 木棍 | 4拳 | 5拳 | 6拳 |
| 石斧 | 3拳 | 3-4拳 | 4-5拳 |
| 铁剑 | 2拳 | 2拳 | 3拳 |

### 2.5 体质减免效果

| 体质 | 减免 | 效果 |
|------|------|------|
| 0.3 | 3 | 容易被击杀 |
| 0.5 | 5 | 中等 |
| 0.8 | 8 | 更难击杀 |

## 3. 实现

### 3.1 Character新增属性

```typescript
class Character {
    // 生命值
    health: number = 100;
    
    // 最大生命值（体质影响）
    get maxHealth(): number {
        return 75 + this.phenotype.constitution * 50;
    }
    
    // 基础伤害（力量影响）
    get baseDamage(): number {
        return 10 + this.phenotype.strength * 20;
    }
    
    // 攻击
    attack(target: Character, weaponBonus: number = 0): number {
        const damage = this.baseDamage + weaponBonus;
        const actualDamage = damage - target.phenotype.constitution * 10;
        target.health -= actualDamage;
        return actualDamage;
    }
}
```

### 3.2 生命值消耗

```typescript
update(deltaTime: number): void {
    // 原有消耗
    this.food = Math.max(0, this.food - consumption);
    this.water = Math.max(0, this.water - consumption * 1.5);
    
    // 生命值消耗
    if (this.water === 0) {
        this.health -= 2 * deltaTime / 60; // 2点/分钟
    }
    if (this.food === 0) {
        this.health -= 1 * deltaTime / 60; // 1点/分钟
    }
    
    // 死亡检查
    if (this.health <= 0) {
        this.die();
    }
}
```

## 4. 死亡处理

### 4.1 死亡条件
- 生命值归0

### 4.2 死亡后果
- 角色消失
- 物品掉落
- 知识传承（如有子女）
