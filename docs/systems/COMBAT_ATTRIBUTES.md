# 核心战斗属性系统

> 版本：v0.11.0
> 日期：2026-03-23
> 状态：开发中

---

## 1. 概述

核心战斗属性控制AI角色在危险情境下的行为决策，包括：
- 胆量（探索危险区域）
- 恐惧阈值（逃跑触发）
- 攻击性（冲突中主动攻击）

---

## 2. 属性定义

### 2.1 胆量 (Bravery)

| 项目 | 值 |
|------|---|
| DNA字段 | `bravery` |
| 范围 | 0.0 - 1.0 |
| 默认值 | 0.5 |
| 影响 | 探索危险区域的意愿 |

**行为影响：**
```
胆量高 (0.7-1.0)
├── 愿意进入高山/洞穴/深海
├── 主动接近野兽
└── 挑战未知区域

胆量中 (0.3-0.7)
├── 有限探索
└── 遇到危险会犹豫

胆量低 (0.0-0.3)
├── 远离危险区域
├── 留守安全地带
└── 群体中跟随他人
```

### 2.2 恐惧阈值 (FearResponse)

| 项目 | 值 |
|------|---|
| DNA字段 | `fearResponse` |
| 范围 | 0.0 - 1.0 |
| 默认值 | 0.5 |
| 影响 | 触发逃跑的惊吓程度 |

**行为影响：**
```
恐惧阈值低 (0.0-0.3) - 易受惊
├── 轻微危险就逃跑
├── 容易被吓到
└── 群体中易恐慌

恐惧阈值中 (0.3-0.7)
├── 中等危险触发
└── 需要明显威胁才逃跑

恐惧阈值高 (0.7-1.0) - 临危不惧
├── 只有重大危险才逃跑
├── 镇定应对
└── 可能选择战斗
```

### 2.3 攻击性 (Aggression)

| 项目 | 值 |
|------|---|
| DNA字段 | `aggression` |
| 范围 | 0.0 - 1.0 |
| 默认值 | 0.5 |
| 影响 | 冲突中的主动攻击倾向 |

**行为影响：**
```
攻击性高 (0.7-1.0)
├── 先发制人
├── 冲突时主动攻击
└── 容易与他人起冲突

攻击性中 (0.3-0.7)
├── 有限攻击
└── 防御为主

攻击性低 (0.0-0.3)
├── 避免冲突
├── 优先逃跑/和解
└── 被动防御
```

## 3. 行为组合

### 3.1 性格矩阵

| 组合 | 胆量 | 恐惧阈值 | 攻击性 | 行为表现 |
|------|------|----------|--------|----------|
| 勇士 | 高 | 高 | 高 | 艺高人胆大，遇险战斗 |
| 莽夫 | 高 | 低 | 高 | 艺高但易受惊，爱打架 |
| 谨慎战士 | 高 | 高 | 低 | 勇敢但能撑住，被动防御 |
| 逃兵 | 低 | 低 | 低 | 怂包，一吓就跑 |
| 旁观者 | 低 | 高 | 低 | 谨慎但不慌，观察为主 |
| 挑衅者 | 中 | 中 | 高 | 爱惹事但会掂量 |

### 3.2 危险情境决策

```
遇到其他角色/野兽
        ↓
    有威胁？
    ↙    ↘
  是      否
  ↓        ↓
胆量足够？  攻击性足够？
↙    ↘    ↙    ↘
是    否  是    否
↓      ↓  ↓      ↓
战斗   恐惧阈值？ 攻击   离开
      ↙    ↘
    是      否
    ↓        ↓
   逃跑    战斗
```

## 4. 实现

### 4.1 DNA Phenotype

```typescript
interface Phenotype {
    // 核心战斗属性
    bravery: number;           // 胆量 (0-1)
    fearResponse: number;      // 恐惧阈值 (0-1)
    aggression: number;         // 攻击性 (0-1)
    
    // 其他战斗相关
    strength: number;          // 力量
    constitution: number;      // 体质
    agility: number;           // 敏捷
}
```

### 4.2 决策方法

```typescript
class Character {
    // 判断是否探索危险区域
    shouldExploreDanger(): boolean {
        return this.phenotype.bravery > 0.6;
    }
    
    // 判断是否逃跑
    shouldFlee(dangerLevel: number): boolean {
        const fearTrigger = dangerLevel > (1 - this.phenotype.fearResponse);
        return fearTrigger;
    }
    
    // 判断是否主动攻击
    shouldAttack(): boolean {
        return this.phenotype.aggression > 0.6;
    }
}
```
