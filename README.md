# 伊甸世界 (Eden World)

> AI文明演化模拟器 - 观察AI角色在世界中自主生存、繁衍、发展

## 🎯 项目简介

伊甸世界是一个基于AI驱动的文明演化模拟器。玩家设定规则，观察AI角色在世界中自主决策、繁衍、发展的过程。

### 核心理念

> **只定义规则，不定义个体选择**
>
> 我们只定义世界的规则，不定义AI的行为选择。AI是自由的个体，他们会自己决定做什么。

## 🎮 游戏素材

本项目使用自制的像素风游戏素材，位于 `img/` 目录：

| 类型 | 数量 | 内容 |
|------|------|------|
| 地形 | 26+ | 草地、平原、森林、沙漠、山地、丘陵、海洋、湖泊、沼泽、河流、沙滩 |
| 物品 | 10+ | 树木、灌木、石头、木棍、树枝、浆果、花朵 |
| 角色 | 2 | 亚当、夏娃 |
| 建筑 | 4 | 窝棚、木屋、石墙、水井 |
| 工具 | 3 | 石斧、石矛、石锤 |

> ⚠️ 素材文件较大，已上传至GitHub，clone后可直接使用。

## 🛠️ 技术栈

- **语言**: TypeScript
- **渲染**: PixiJS v8（2D渲染引擎，GPU加速）
- **运行时**: Node.js + 浏览器

## 📦 安装

```bash
npm install
```

## 🚀 运行

```bash
# 开发模式
npm run dev

# 或直接启动服务器
npm run preview
```

然后访问 http://localhost:3000

## 🎮 操作说明

- **WASD / 方向键**: 移动相机
- **鼠标拖拽**: 移动地图
- **滚轮**: 缩放地图
- **1/2/3**: 切换缩放级别

## 📁 项目结构

```
/eden-world
  /src
    /core           # 核心引擎
    /world          # 世界相关（地图生成）
    /renderer       # 渲染系统（PixiJS）
    /systems        # 游戏系统
    │   /dna        # DNA系统（遗传与表现型）
    │   /needs      # 需求系统（饥饿/口渴/安全等）
    │   /ai         # AI决策系统
    /entities       # 实体
    /utils          # 工具函数
  /docs             # 文档
  /tests            # 测试
  /img              # 游戏素材
```
