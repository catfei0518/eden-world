# 伊甸世界 (Eden World)

> AI文明演化模拟器 - 观察AI角色在世界中自主生存、繁衍、发展

## 🎯 项目简介

伊甸世界是一个基于AI驱动的文明演化模拟器。玩家设定规则，观察AI角色在世界中自主决策、繁衍、发展的过程。

### 核心理念

> **只定义规则，不定义个体选择**
> 
> 我们只定义世界的规则，不定义AI的行为选择。AI是自由的个体，他们会自己决定做什么。

## 📖 功能特点

- 🌟 完整的AI认知架构
- 🌍 程序化生成的无限世界
- ⚔️ 涌现的社会现象
- 🎨 可视化界面

## 🛠️ 技术栈

- **语言**: TypeScript
- **渲染**: PixiJS
- **运行时**: Node.js

## 📦 安装

```bash
npm install
```

## 🚀 快速开始

```bash
npm run dev
```
