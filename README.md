# 伊甸世界 (Eden World)

> AI文明演化模拟器 - 观察AI角色在世界中自主生存、繁衍、发展

## 🎯 项目简介

伊甸世界是一个基于AI驱动的文明演化模拟器。玩家设定规则，观察AI角色在世界中自主决策、繁衍、发展的过程。

### 核心理念

> **只定义规则，不定义个体选择**
>
> 我们只定义世界的规则，不定义AI的行为选择。AI是自由的个体，他们会自己决定做什么。

## 🛠️ 技术栈

- **语言**: TypeScript
- **渲染**: PixiJS v8（2D渲染引擎，GPU加速）
- **运行时**: Node.js + 浏览器

## 📦 安装

```bash
npm install
```

## 🚀 运行

```bash
# 开发模式
npm run dev

# 或直接启动服务器（端口3333）
npm run preview
```

然后访问 http://localhost:3333

## 🎮 操作说明

- **WASD / 方向键**: 移动相机
- **鼠标拖拽**: 移动地图
- **滚轮**: 缩放地图
- **1/2/3**: 切换缩放级别
- **`~` 或 `/`**: 打开控制台

## 📁 项目结构

```
/eden-world
  /src
    /core           # 核心引擎
    /world          # 世界相关（地图生成）
    /renderer       # 渲染系统（PixiJS）
    │   /pixi
    │       /layers # 渲染层（地形/物品/角色）
    │   StatusUI.ts      # 角色状态面板
    │   ItemStatusUI.ts  # 物品信息面板
    │   ConsoleUI.ts    # 控制台界面
    /systems        # 游戏系统
    │   /dna        # DNA系统（遗传与表现型）
    │   /needs      # 需求系统（饥饿/口渴/安全等）
    │   /ai         # AI决策系统
    │   CommandSystem.ts # 命令系统（支持自定义指令）
    /entities       # 实体（角色等）
  /docs             # 文档
  /tests            # 测试
  /img              # 游戏素材
```
